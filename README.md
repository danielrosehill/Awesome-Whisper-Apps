# Awesome Whisper Apps

[OpenAI Whisper](https://github.com/openai/whisper) is a robust automatic speech recognition (ASR) system trained on 680,000 hours of multilingual and multitask supervised data collected from the web.

## Official & Source

- [OpenAI Whisper](https://github.com/openai/whisper) - Official OpenAI Whisper repository
- [Whisper on Hugging Face](https://huggingface.co/collections/openai/whisper-release) - Official Whisper model collection on Hugging Face

### Official Hugging Face Models

- [whisper-large](https://huggingface.co/openai/whisper-large) - Largest model with best accuracy
- [whisper-medium](https://huggingface.co/openai/whisper-medium) - Medium-sized model balancing accuracy and speed
- [whisper-small](https://huggingface.co/openai/whisper-small) - Small model for faster inference
- [whisper-base](https://huggingface.co/openai/whisper-base) - Base model for resource-constrained environments
- [whisper-tiny](https://huggingface.co/openai/whisper-tiny) - Smallest model for minimal resource usage
